# 주의사항
- 카페에 댓글이 아닌 답글로 작성
- 답글의 제목은 "[1A. 과제2]"등 과제번호의 앞부분을 그대로 제목에 사용
- 과제 제목을 ```반_학번_이름``` 으로 변경
```
[1A. 과제2] A_000000_홍길동
```
- 댓글로 달지말고, 답글로 달기
- 파일로 올리지 말고, 동영상으로 올리기
